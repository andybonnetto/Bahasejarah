**Interactive Historical Language Map of Indonesia**

Bahasejarah is an interactive web application that visualizes the historical distribution and evolution of languages across the Indonesian archipelago. Users can explore different regions, view timelines of language development, and learn about the scripts and history of various local languages.

- **[Omniglot](https://www.omniglot.com)**: Used as the primary reference for alphabet and script images.
